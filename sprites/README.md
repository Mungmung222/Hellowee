# 🎨 스프라이트 폴더 구조

## 레이어 기반 커스텀 시스템

각 파츠를 폴더별로 정리합니다.

```
sprites/
├── body/
│   ├── default/
│   │   ├── idle.png      ← 128×128 × 2프레임 = 가로 256px
│   │   ├── walk.png      ← 128×128 × 3프레임 = 가로 384px
│   │   ├── fall.png      ← 128×128 × 1프레임 = 가로 128px
│   │   ├── grabbed.png   ← 128×128 × 1프레임 = 가로 128px
│   │   ├── land.png      ← 128×128 × 1프레임 = 가로 128px
│   │   └── thrown.png    ← 128×128 × 1프레임 = 가로 128px
│   ├── light/
│   │   └── (동일 구조)
│   ├── medium/
│   └── dark/
├── eyes/
│   ├── round/
│   ├── cat/
│   └── sleepy/
├── mouth/
│   ├── smile/
│   ├── neutral/
│   └── pout/
├── hair/
│   ├── short/
│   ├── long/
│   └── ponytail/
├── clothes/
│   ├── hoodie/
│   ├── shirt/
│   └── dress/
└── accessory/
    ├── hat/
    ├── glasses/
    └── ribbon/
```

## 규칙

1. 모든 파츠 PNG는 **128×128px 캔버스 기준**
2. 프레임은 **가로로** 이어붙인 스프라이트시트
3. 모든 파츠가 **같은 위치 기준**으로 그려져야 함 (겹쳤을 때 맞도록)
4. 투명배경 PNG
5. 좌우반전은 코드에서 처리하므로 **오른쪽 보는 방향만** 그리면 됨

## 동작별 프레임 수

| 동작 | 프레임 수 |
|------|----------|
| idle | 2 |
| walk | 3 |
| fall | 1 |
| grabbed | 1 |
| land | 1 |
| thrown | 1 |

## 그리기 팁

- idle 기본 포즈 하나 잘 그리고, 나머지는 복사해서 변형
- 피부색은 같은 그림에서 색상만 변경
- 눈/입은 위치만 맞으면 몸통과 겹쳐도 자연스러움
