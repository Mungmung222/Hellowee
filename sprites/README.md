# 스프라이트 폴더

여기에 캐릭터 이미지 파일들을 넣으세요.

## 필수 파일
- `idle.png` — 스프라이트시트, 64×64px × 4프레임 (가로 256px)
- `walk.png` — 스프라이트시트, 64×64px × 6프레임 (가로 384px)
- `fall.png` — 스프라이트시트, 64×64px × 2프레임 (가로 128px)
- `grabbed.png` — 단일 이미지, 64×64px
- `land.png` — 스프라이트시트, 64×64px × 2프레임 (가로 128px)

## 스프라이트시트 형식
모든 프레임을 가로로 이어붙인 PNG 파일
[프레임1][프레임2][프레임3]...

이미지가 없으면 임시로 🐾 박스가 표시됩니다.
