# 🐾 Hellowee — 전체 기획서 v2.1

## 📌 컨셉
멀티플레이어 시메지 (바탕화면 위 캐릭터 앱)
친구들끼리 코드로 방 만들어서 캐릭터가 서로 바탕화면에서 돌아다니며 소통

---

## 🖥️ 플랫폼
| 플랫폼 | 기술 | 배경 위 캐릭터 |
|--------|------|---------------|
| 맥/윈도우 | Electron | ✅ |
| 갤럭시 | React Native + 오버레이 | ✅ |
| 아이폰 | React Native | ❌ 앱 내에서만 |

---

## 🔐 로그인
- 소셜 로그인: 구글 / 카카오 / 네이버 (Supabase Auth)
- 첫 로그인: 닉네임 + 약관 동의
- **자동 로그인 저장** (electron-store)
- 같은 계정 = 같은 캐릭터

---

## 🏠 방 시스템
- 방 만들기 → 방 이름 + 6자리 코드 (중복 불가)
- 입장: 방 이름 + 코드 매칭
- 방장 오프라인이어도 입장 가능
- 종료 후 재입장 가능 (강퇴 안 당한 한)
- 방장: 강퇴, 방 삭제

---

## 📱 UI — 핸드폰 (홈화면 아이콘형)

내 캐릭터 더블클릭 → 핸드폰 UI

### 홈 화면
```
💬 채팅    🏠 로비
🎨 꾸미기   ⚙️ 설정
```

### 각 페이지
- 💬 채팅: 전체/귓속말 탭 + 입력
- 🏠 로비: 현재방 / 방만들기 / 입장 / 코드복사 / 강퇴
- 🎨 꾸미기: 파츠 선택 + 프리뷰
- ⚙️ 설정: 핀/어텐션/말풍선/소리 토글, 닉네임 변경, 튜토리얼 다시보기, 방나가기

---

## 🖱️ 인터랙션

| 대상 | 동작 | 결과 |
|------|------|------|
| 내 캐릭터 | 1번 클릭 | 랜덤 모션 + 빈 💬 말풍선 |
| 빈 말풍선 | 클릭 | 퀵 전체채팅 |
| 내 캐릭터 | 더블클릭 | 폰 UI |
| 느린 드래그 | 놓기 | 떨구기 |
| 빠른 드래그 | 놓기 | 던지기 |
| 상대 캐릭터 | 1번 클릭 | 빈 🤫 말풍선 |
| 빈 말풍선 | 클릭 | 퀵 귓속말 |
| 근접 (40px) | 자동 | 서로 인사 |

## 📖 튜토리얼

### 시점
- **첫 실행 시 자동** (로그인 + 닉네임 설정 직후)
- 이후: ⚙️ 설정 → "📖 튜토리얼 다시보기" 버튼

### 형태
- 캐릭터 옆 **가이드 말풍선** + **하이라이트 영역**
- 스텝별로 진행 (다음/건너뛰기 버튼)
- 반투명 오버레이로 현재 설명 부분만 밝게

### 스텝 구성

| 스텝 | 화면 | 가이드 말풍선 내용 | 유저 액션 |
|------|------|------------------|----------|
| 1 | 캐릭터 등장 | "안녕! 나는 너의 Hellowee 캐릭터야 🐾" | [다음] |
| 2 | 캐릭터 하이라이트 | "나를 한번 클릭해봐!" | 유저가 캐릭터 클릭 |
| 3 | 빈 말풍선 뜸 | "이 말풍선을 누르면 모두에게 메시지를 보낼 수 있어!" | 유저가 말풍선 클릭 |
| 4 | 퀵채팅 입력창 | "여기에 메시지를 적고 Enter! 💬" | [다음] |
| 5 | 캐릭터 하이라이트 | "이번엔 나를 빠르게 두 번 클릭해봐!" | 유저가 더블클릭 |
| 6 | 폰 UI 열림 | "짠! 이게 너의 Hellowee 폰이야 📱" | [다음] |
| 7 | 홈 아이콘 하이라이트 | "💬채팅 🏠로비 🎨꾸미기 ⚙️설정 — 여기서 다 할 수 있어!" | [다음] |
| 8 | 로비 아이콘 하이라이트 | "🏠 로비에서 방을 만들거나 친구 방에 입장할 수 있어" | [다음] |
| 9 | 캐릭터 드래그 안내 | "나를 드래그하면 끌고다닐 수 있어! 빠르게 놓으면 날아가 😆" | 유저가 드래그 |
| 10 | 다른 캐릭터 표시 | "친구 캐릭터를 클릭하면 귓속말을 보낼 수 있어 🤫" | [다음] |
| 11 | 완료 | "준비 완료! 이제 친구들이랑 놀자 🐾🎉" | [시작하기] |

### 건너뛰기
- 모든 스텝에서 "건너뛰기" 가능
- 건너뛰면 즉시 로비 화면으로

### 기술 구현
- `localStorage`에 `hellowee_tutorial_done = true` 저장
- 설정에서 "튜토리얼 다시보기" 누르면 → 플래그 리셋 후 재시작
- 튜토리얼 중에는 실제 서버 연결 안 함 (오프라인 연습)
- 스텝 9~10은 AI 더미 캐릭터로 시뮬레이션
