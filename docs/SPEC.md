# 🐾 Hellowee — 전체 기획서 v2

## 📌 컨셉
멀티플레이어 시메지 (바탕화면 위 캐릭터 앱)
친구들끼리 코드로 방 만들어서 캐릭터가 서로 바탕화면에서 돌아다니며 소통

---

## 🖥️ 플랫폼
| 플랫폼 | 기술 | 배경 위 캐릭터 |
|--------|------|---------------|
| 맥/윈도우 | Electron | ✅ |
| 갤럭시 | React Native + 오버레이 | ✅ |
| 아이폰 | React Native | ❌ 앱 내에서만 |

---

## 🔐 로그인
- 소셜 로그인: 구글 / 카카오 / 네이버 (Supabase Auth)
- 첫 로그인: 닉네임 + 약관 동의
- **자동 로그인 저장** (electron-store)
- 같은 계정 = 같은 캐릭터

---

## 🏠 방 시스템
- 방 만들기 → 방 이름 + 6자리 코드 (중복 불가)
- 입장: 방 이름 + 코드 매칭
- 방장 오프라인이어도 입장 가능
- 종료 후 재입장 가능 (강퇴 안 당한 한)
- 방장: 강퇴, 방 삭제

---

## 📱 UI — 핸드폰 (홈화면 아이콘형)

내 캐릭터 더블클릭 → 핸드폰 UI

### 홈 화면
```
💬 채팅    🏠 로비
🎨 꾸미기   ⚙️ 설정
```

### 각 페이지
- 💬 채팅: 전체/귓속말 탭 + 입력
- 🏠 로비: 현재방 / 방만들기 / 입장 / 코드복사 / 강퇴
- 🎨 꾸미기: 파츠 선택 + 프리뷰
- ⚙️ 설정: 핀/어텐션/말풍선/소리 토글, 닉네임 변경, 방나가기

---

## 🖱️ 인터랙션

| 대상 | 동작 | 결과 |
|------|------|------|
| 내 캐릭터 | 1번 클릭 | 랜덤 모션 + 빈 💬 말풍선 |
| 빈 말풍선 | 클릭 | 퀵 전체채팅 |
| 내 캐릭터 | 더블클릭 | 폰 UI |
| 느린 드래그 | 놓기 | 떨구기 |
| 빠른 드래그 | 놓기 | 던지기 |
| 상대 캐릭터 | 1번 클릭 | 빈 🤫 말풍선 |
| 빈 말풍선 | 클릭 | 퀵 귓속말 |
| 근접 (40px) | 자동 | 서로 인사 |
