# 🐾 Hellowee — 전체 기획서 v3.0 (Final)

> 멀티플레이어 시메지 — 친구들이랑 바탕화면에서 노는 데스크탑 캐릭터 앱

## 완료된 기능 요약

### 플랫폼
- 데스크탑: Electron (맥/윈도우)
- 모바일: React Native (예정)
- 서버: Node.js + Socket.io + Supabase

### 로그인
- 구글/카카오/네이버 소셜 로그인
- 게스트 모드 (로그인 없이)
- 자동 로그인 저장 (electron-store)
- 닉네임 + 약관 동의

### 방 시스템
- 방 이름 + 6자리 코드 (둘 다 매칭해야 입장)
- 방장 오프라인에도 입장 가능
- 종료 후 재입장 / 마지막 방 자동 재접속
- 방장: 강퇴 + 위임
- 초대링크 복사 (방이름+코드 한번에)

### 인터랙션
- 1번 클릭: 랜덤 모션 + 빈 말풍선 → 퀵 채팅
- 더블클릭: 폰 UI
- 느린 드래그: 떨구기
- 빠른 드래그: 던지기
- 캐릭터 근접: 자동 인사

### 채팅
- 전체 채팅 (흰 말풍선) + 귓속말 (보라 말풍선)
- 이모지 빠른 전송
- 핸드폰 UI 기록창

### 어텐션 모드
- 데스크탑: 마우스 쫓아가기 + 3초 idle 시 달라붙기
- 모바일: 방방뛰기 + 배너

### UI
- 홈화면 아이콘 그리드 (4개: 채팅/로비/꾸미기/설정)
- 뒤로가기 네비게이션
- 닉네임 변경 모달
- 상태 메시지 (🟢온라인/🟡자리비움/🔴일하는중 등)

### 기타
- 시스템 트레이 (백그라운드 실행)
- 튜토리얼 시스템 (10단계)
- 합성 사운드 효과음 12종
- 종료 시 손흔들기 애니메이션
- 레이어 기반 캐릭터 커스텀
